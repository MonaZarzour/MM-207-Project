# Data Privacy Policy

## What data we collect

We collect only the minimum data required to operate the service:

- Email address
- Display name (optional)
- Password (stored as a secure hash)
- Terms of Service consent timestamp and version
- Technical session identifiers

## Why we collect this data

The data is used to:

- Create and manage user accounts
- Authenticate users
- Enable sharing and collaboration on lists
- Maintain service security and integrity

## Data minimization & limitations

This service uses in-memory storage and does not persist data to a database.
All user data is lost if the server restarts.
We do not collect unnecessary personal information.

## Security

Passwords are never stored in plain text.
Authentication is handled using secure tokens.
In a production environment, all communication would be encrypted using HTTPS.

## User rights

Users have the right to:

- Access their personal data
- Delete their account at any time
- Withdraw consent by deleting their account

When an account is deleted, all personal data is removed.
Public contributions (such as shared lists or items) may remain, but are anonymized.

## Consent

Creating an account requires explicit consent to the Terms of Service.
Without consent, account creation is not possible.
