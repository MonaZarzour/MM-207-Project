Data Privacy Policy

1. Introduction

This Privacy Policy describes how the Shared Shopping List application processes personal data.

The application is developed for academic purposes and is not intended for commercial use.

2. Data We Collect

We collect only the minimum data necessary to operate the service:

Email address

Display name (optional)

Password (stored as a secure hash)

Terms of Service consent status, timestamp, and version

Technical session or authentication identifiers

No additional personal data is collected.

3. Purpose of Data Collection

The collected data is used exclusively to:

Create and manage user accounts

Authenticate users

Enable list sharing and collaboration

Enforce access control and authorization

Maintain service integrity and security

Data is not used for marketing, profiling, or third-party sharing.

4. Data Storage & Retention

This application uses in-memory storage and does not persist data in a database.

All stored data is temporary and will be lost when the server restarts.

No long-term retention of personal data occurs in this academic environment.

5. Security Measures

Passwords are never stored in plain text.

Passwords are stored using secure hashing mechanisms.

Authentication is handled using secure tokens.

In a production environment, communication would be encrypted using HTTPS.

6. User Rights

Users have the right to:

Access their personal data

Delete their account at any time

Withdraw consent by deleting their account

When an account is deleted:

All associated personal data is removed.

Shared list data may remain but is anonymized where applicable.

7. Consent

Creating an account requires explicit acceptance of the Terms of Service.

Account creation is rejected if consent is not provided.

Consent status is stored on the server.
